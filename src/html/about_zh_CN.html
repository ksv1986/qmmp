基于Qt的多媒体播放器(Qmmp)。

此程序是一个音乐播放器，程序的编写基于Qt库。

主要功能：

  • 未压缩的winamp皮肤支持；
  • 插件支持；
  • MPEG1 1/2/3 层支持；
  • Ogg Vorbis 支持；
  • native FLAC 支持；
  • Musepack 支持；
  • WMA 支持；
  • ALSA 声音输出；
  • JACK 声音输出。

要求：

  • OS GNU Linux;
  • Qt версии >= 4.2;
  • MAD;
  • Ogg Vorbis;
  • FLAC >= 1.1.3;
  • ALSA >= 1.0.1;
  • TagLib >= 1.4.
  • libmpcdec >= 1.2.6
  • Jack >= 0.102.5
  • FFmpeg >= 0.4.9-pre1
