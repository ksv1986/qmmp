Qt-based Multimedia Player (Qmmp)

Qmmp je přehrávač zvuku založený na knihovně Qt.

Klíčové vlastnosti:

  • podpora témat programu Winamp;
  • podpora zásuvných modulů (plugin);
  • podpora MPEG1 layer 1/2/3;
  • podpora Ogg-Vorbis;
  • podpora FLAC;
  • podpora Musepack;
  • podpora WMA;
  • zvukový výstup ALSA;
  • zvukový výstup JACK.

Požadavky:

  • operační systém Linux;
  • Qt >= 4.2;
  • MAD;
  • Ogg-Vorbis;
  • FLAC >= 1.1.3;
  • ALSA >= 1.0.1;
  • TagLib >= 1.4.
  • libmpcdec >= 1.2.6
  • Jack >= 0.102.5
  • FFmpeg >= 0.4.9-pre1
